## Workflow de mise à jour

1. Modifier le fichier voulu (par ex. `poulet-gifle-profbucella/index.html`)
2. Dans GitHub Desktop : commit + push
3. Netlify détecte le push, redéploie automatiquement le site concerné en 30 s
